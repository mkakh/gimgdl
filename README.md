# Google画像検索から自動で画像を保存するスクリプト

Google画像検索の仕様上，300枚程度しかダウンロードできないため，bing画像検索を使うことをおすすめします．

[Bing画像検索から自動で画像を保存するスクリプト](https://github.com/mkakh/bimgdl/blob/master/bimgdl.ipynb)

# Usage
```
python gimgdl 検索語
```
または，jupyter-notebookで開いて使う
